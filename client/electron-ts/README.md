# Electron client written in typescript

#### Dependencies
- webpack (4.29.3)
- typescript
- dotenv

#### Set up 
`npm install`    
`npm run build`    
`npm run start`    
client will start.     
    
check ./src/config/config.ts 

#### environment
this project uses dotenv so you can copy .env.example to .env.
Then, env file will be loaded via `dotenv`, see ./app/main.js
